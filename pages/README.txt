This site is built as a single scrolling page (index.html) with all 11 "pages" from
the blueprint implemented as sections: Home, Timeline, Reasons, Gallery, Love Letter,
Gift Box, Cake, Night Sky, and Final Wishes (plus the Loading Screen and Welcome Popup
as full-screen overlays). This is the standard, more polished pattern for sites like
this — smooth scrolling between "pages" instead of full reloads.

If you'd rather split any section into its own real .html file, copy the section's
markup out of index.html into a file here and link to it from the nav.
